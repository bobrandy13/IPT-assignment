    what is an object????

They can also have functoins as part of the object. For example, humans can walk, talk, run. Its more of a way to bundle them together.

    • Different objects can have different attributes

                
                class Human {
                    constructor(name: string, age: number) {
                        this.name = name; <-- This is an attribute
                        this.age = age; <-- so is this one
                    }
                }

    For example, a dog can have a name, age, breed, and color. A human can have a name, age, height, weight, and hair color. These are all different attributes that are unique to the object.

  • Methods/Operations

                
                class Human {
                    constructor(name: string, age: number) {
                        this.name = name;
                        this.age = age;
                    }
                    // this below is a method. It is a function that is part of the object that does something
                    talk() { 
                        console.log(`${this.name} is yapping!`);`
                    }
                }

                const myBestFriend = new Human("Avi Gupta", 30000);
                myBestFriend.talk(); <-- Output : "Avi Gupta is yapping!"
